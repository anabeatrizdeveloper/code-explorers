How Do Computers Think?

Computers follow rules and instructions step by step.

Computers are very literal

A computer does exactly what it is told. If the instructions are wrong, the computer will still follow them.

Computers use logic

Logic means making decisions with rules.

For example:

  • If it is raining, take an umbrella.
  • If you are hungry, eat a snack.

Computers use this same idea in code.

Example

Imagine a game character.

  • If it touches a coin, score +1.
  • If it touches water, stop moving.

That is computer thinking with simple rules.
