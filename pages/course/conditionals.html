What Are Conditionals?

Conditionals help computers make decisions.

Making choices

A conditional tells a computer what to do if something happens.

It is like saying: If this is true, do this.

Example

If it rains, take an umbrella.

If the player touches a coin, add 1 point.

Computers use rules like these to make choices in games and apps.

Why are conditionals important?

They help programs react to what is happening. This makes websites, games, and apps smarter and more interactive.

Explorer Fact

In a game, a character might lose a life if it touches fire. That is a conditional.
