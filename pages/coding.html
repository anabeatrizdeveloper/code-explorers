What is Coding?

Coding means giving instructions to a computer. A computer cannot guess, so we need to tell it exactly what to do.


A simple example

Imagine telling a robot how to brush its teeth: pick up toothbrush, add toothpaste, brush teeth, rinse. That is a list of instructions. That is a lot like coding.


Why is coding important?

Coding helps people create games, apps, websites, robots, and many other cool things.
